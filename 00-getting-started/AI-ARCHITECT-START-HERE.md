## üõ†Ô∏è Using AI Coding Agents

### We Recommend Claude Code

**Why Claude Code:**
- Long context (200K tokens) - sees your whole system
- Strong at architecture and reasoning
- Good at following patterns
- Works well with our workflows

**But you can use anything:**
- Cursor (fast iterations)
- GitHub Copilot (familiar if you use VS Code)
- Codex (via API)
- ChatGPT (copy-paste)
- Manual coding (full control)

**Our workflows are written for Claude Code, but adapt easily to any tool.**

### How to Use the Workflows

1. Open pattern's `AGENT_WORKFLOW.md`
2. Read the architecture first
3. Copy prompts into Claude Code (or your tool)
4. Build step by step
5. Test and deploy

**Example: Building RAG Chatbot**
```
Time: 30 minutes with Claude Code
Steps: 5 phases (setup, backend, frontend, polish, deploy)
Result: Working chatbot with vector search
```
