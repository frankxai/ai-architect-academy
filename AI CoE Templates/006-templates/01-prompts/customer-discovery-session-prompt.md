# Customer Discovery Session Prompt

**Use Case**: Prepare structured discovery questions for a customer engagement

## Prompt Structure:
```
Using the discovery-questions-template.md, create a tailored discovery questionnaire for:

**Customer Profile:**
- Customer Name: [Customer organization]
- Industry Vertical: [Specific industry]
- Organization Size: [Enterprise/Mid-market/SMB]
- Technical Maturity: [Advanced/Intermediate/Basic]

**Engagement Context:**
- Pattern Focus: [Design Pattern XX - Pattern Name]
- Primary Stakeholder: [CTO/IT Director/Business Leader]
- Engagement Stage: [Initial/Qualified/Proof of Concept]
- Timeline Sensitivity: [Urgent/Normal/Exploratory]

**Known Information:**
- Current Challenge: [What we know about their challenges]
- Existing Technology: [Known current technology stack]
- Business Drivers: [Known business motivations]
- Constraints: [Known limitations or requirements]

**Discovery Objectives:**
- Qualification Goals: [Budget, timeline, decision-makers]
- Technical Scoping: [Architecture requirements, integrations]
- Business Alignment: [Value proposition, success criteria]
- Risk Assessment: [Potential implementation challenges]

Generate a comprehensive discovery questionnaire that:
1. Progresses logically from current state to future vision
2. Includes follow-up probes for deeper understanding
3. Qualifies the opportunity while building rapport
4. Gathers technical requirements for accurate scoping
5. Can be used by sales engineers or technical account managers
6. Produces actionable information for solution design
```

## Example Usage Scenario

**Scenario**: Discovery Session Preparation  
A technical account manager is preparing for a discovery call with a retail customer exploring customer analytics capabilities.

**Application**: Use this prompt with retail context, analytics focus, and customer experience improvement objectives to prepare targeted discovery questions.
