# Automation Notes

## Scripts
- `npm run ops:daily` - Create a daily research brief
- `npm run ops:weekly` - Create a weekly report
- `npm run ops:brief -- --type=concept --slug=agentic-rag` - Create a content brief

## Output Directories
- `ops/daily-briefs/`
- `ops/weekly-briefs/`
- `ops/content-briefs/`

## Conventions
- Dates use ISO format (YYYY-MM-DD).
- Slugs are lower-case, hyphenated.
