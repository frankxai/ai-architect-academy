# Review Rubric (Human + AI)

Score each category 0-2. Target score: 16+.

## 1) Direct Answer
0: Buried or missing answer
1: Present but unclear
2: Clear answer in first paragraph

## 2) Human Clarity
0: Dense or jargon-heavy
1: Mostly clear
2: Simple, scannable, actionable

## 3) Technical Accuracy
0: Unverified claims
1: Mostly accurate
2: Verified, testable, sourced

## 4) Examples + Proof
0: No examples
1: Examples but thin
2: Concrete, implementable examples

## 5) Structure for AI
0: No schema or poor headings
1: Some structure
2: Proper headings + JSON-LD

## 6) Internal Linking
0: No links
1: Minimal links
2: Strategic links to related content

## 7) Freshness
0: No date or outdated
1: Date present
2: Date + recent references

## 8) Visual Aids
0: None
1: One simple visual
2: Useful diagram or table
