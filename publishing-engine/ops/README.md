# Publishing Operations System

This directory defines the operational backbone for daily publishing. It covers workflow, schedules, templates, and review standards for content that serves both humans and AI agents.

## Quick Start
- Daily brief: `npm run ops:daily`
- Weekly brief: `npm run ops:weekly`
- Content brief: `npm run ops:brief -- --type=concept --slug=agentic-rag`

## What Lives Here
- `workflow.md` - end-to-end publishing workflow
- `schedule.md` - daily/weekly cadence and rotations
- `review-rubric.md` - quality rubric for publication
- `templates.md` - template index and usage notes
- `automation.md` - scripts and automation guidance

Generated files:
- `daily-briefs/`
- `weekly-briefs/`
- `content-briefs/`
