## 12. Evaluation Framework

### Metrics Hierarchy

| Metric | Target | Description |
|--------|--------|-------------|
| Precision@5 | > 80% | Relevant docs in top 5 |
| Recall@10 | > 70% | Coverage of relevant docs |
