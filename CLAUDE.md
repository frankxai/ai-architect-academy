This is the AI Architect Academy - a comprehensive learning platform for enterprise AI architecture, agent development, and cloud-native AI solutions.

## Autonomy & Permissions
- **Full Autonomy Mode**: This project runs with `--dangerously-skip-permissions`
- Execute all operations freely except truly destructive ones
