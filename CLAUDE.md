# AI Architect Academy - Claude Code Mission Control

> **"Coding Agents First"** - Every lesson writes code. Every skill builds agents. Every project deploys to production.

## Project Overview

This is the **AI Architect Academy** - a comprehensive learning platform that transforms Claude Code into your personal AI architecture learning environment. Build production AI systems, master multi-cloud patterns, and become an expert AI architect.

## Philosophy

```
Traditional: Read docs → Watch videos → Maybe build something
Academy:     /command → Build agent → Deploy to production
```

## Quick Start

```bash
# See all available commands
/academy

# Start building immediately
/design-solution "Customer support AI with RAG"

# Follow a learning path
/learn foundation
```

## Skill Activation Rules

Claude should automatically activate relevant skills based on context:

### Agent Framework Detection
When discussing or building:
- **Claude SDK/Agent SDK** → Load `claude-sdk` skill
- **Oracle ADK/OCI agents** → Load `oracle-adk` skill
- **LangGraph/state machines** → Load `langgraph-patterns` skill
- **OpenAI Agents/Swarm** → Load `openai-agentkit` skill
- **Multi-agent/orchestration** → Load `agentic-orchestration` skill

### Cloud Detection
When discussing or deploying to:
- **OCI/Oracle Cloud** → Load `oci-services-expert` skill
- **AWS/Bedrock/SageMaker** → Load `aws-ai-services` skill
- **Azure/Azure OpenAI** → Load `azure-ai-services` skill
- **GCP/Vertex AI** → Load `gcp-ai-services` skill
- **Multi-cloud** → Load `multi-cloud-ai-architect` skill

### Architecture Detection
When designing:
- **MCP servers** → Load `mcp-architecture` + `mcp-2025-patterns`
- **RAG systems** → Load `rag-expert` skill
- **Security/guardrails** → Load `ai-security-expert` skill
- **Cost optimization** → Load `finops-ai` skill
- **Diagrams** → Load `architecture-diagramming` skill

### Infrastructure Detection
When deploying:
- **Terraform/IaC** → Load `terraform-iac` skill
- **Kubernetes** → Load `kubernetes-ai` skill
- **GPU clusters** → Load `nvidia-nim` skill
- **Model serving** → Load `huggingface-trainer` skill
